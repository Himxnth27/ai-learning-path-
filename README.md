# AI-Powered Personalized Learning Path

A personalized learning path generator for Computer Science students, featuring an AI-simulated curriculum builder and progress tracking.

## Features
- **Personalized Paths**: Generates courses for Web Dev, DSA, AI/ML, and more.
- **Resource Integration**: Curated, concise YouTube tutorials for every module.
- **Progress Tracking**: Log into the dashboard to save your journey.
- **Premium UI**: Dark mode, glassmorphism design, and smooth animations.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express
- **Database**: SQLite

## How to Run Locally
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   node server.js
   ```
4. Open `http://localhost:3000/login.html` in your browser.
